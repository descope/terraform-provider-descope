
AdminPortalWidget
=================



widget_id
---------

- Type: `string` (required)

The unique identifier of the Widget



type
----

- Type: `string` (required)

The type of the Widget





AdminPortal
===========



enabled
-------

- Type: `bool` 

Whether the Admin Portal is enabled



style_id
--------

- Type: `string` 

The style id to use



widgets
-------

- Type: `list` of `adminportal.AdminPortalWidget` 

The widgets to show in the Admin Portal
